# Crypto Trading Bot (Python)

A simplified trading bot built in Python to demonstrate:
- Order handling (Market & Limit)
- Input validation
- Logging
- API-like architecture using a simulated Binance client
- CLI interface

This project does NOT require real API keys.  
All orders are simulated for learning and testing purposes.

---

## Features
- Market Order
- Limit Order
- CLI user interface
- Logging system (trading_bot.log)
- Error handling
- Clean object-oriented design

---

## Skills Demonstrated
- Python OOP
- CLI apps
- Logging
- Data classes
- Handling user input
- Simulation of API behaviour
